# GrayJay plugin DRTV

This is a plugin for GrayJay that enables you to watch series available on
dr.dk/drtv.

## Installation

Go to Sources and add a source. Select "Install by URL" or "Install by QR" and
paste the below URL or scan the below QR code respectively.

Install by URL:

```txt
https://raw.githubusercontent.com/TheColorman/grayjay-plugin-drtv/refs/heads/main/DRTVConfig.json
```
